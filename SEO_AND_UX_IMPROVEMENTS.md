### 4. **Open Graph Tags** (Social Media)
```html
<meta property="og:title" content="Explain Like I'm in 5th Grade" />
<meta property="og:description" content="Get any topic explained simply" />
<meta property="og:url" content="https://www.classmate.info" />
<meta property="og:type" content="website" />
```

**Result:** Beautiful previews when shared on Facebook, LinkedIn, etc.

### 5. **Twitter Card**
```html
<meta name="twitter:card" content="summary_large_image" />
<meta name="twitter:title" content="Explain Like I'm in 5th Grade" />
```

**Result:** Rich cards when shared on Twitter/X

### 6. **Structured Data (Schema.org)**
```json
{
  "@context": "https://schema.org",
  "@type": "WebApplication",
  "name": "ClassMate.info - Explain Like I'm in 5th Grade",
  "applicationCategory": "EducationalApplication",
  "offers": {
    "price": "0",
    "priceCurrency": "USD"
  }
}
```

**Result:** Google understands your app better, may show rich results

### 7. **Robots Meta**
```html
<meta name="robots" content="index, follow" />
<meta name="googlebot" content="index, follow, max-snippet:-1" />
```

**Result:** Google can fully index and display your content
